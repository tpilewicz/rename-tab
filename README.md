# Rename Tab

Vibe-coded Firefox extension that lets you rename any browser tab.

## Features

- Click the toolbar icon to rename the current tab via a popup
- Use a keyboard shortcut (**Alt+Shift+R**) to rename inline without leaving the page
- Reset a tab to its original title
- Prevents pages from overriding your custom title

## Usage

**Via popup:** Click the extension icon in the toolbar, type a new title, and press Enter or click Rename.

**Via shortcut:** Press **Alt+Shift+R** to open an inline prompt on the page. Press Enter to confirm or Escape to cancel.

The shortcut can be customized in Firefox at `about:addons` → gear icon → **Manage Extension Shortcuts**.
